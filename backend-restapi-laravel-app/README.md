# Portfolio App- Admin Panel
This is a admin panel of portfolio app and in this admin panel i developed using react-laravel-scaffolding.

### Module Overview <hr>
* Login
* Dashboard
* Services
    * Add New Service
    * All Service List 
    * Service Edit and Update
    * Delete Service
* Contact
    * All Contact List
    * Delete Contact List
* Sign Out

##### Rest Api Developed using Laravel for this module.
* Home Page
* Service Page
* Course Page
* Portfolio / Project Page
* About Page
* Contact Page
* Terms & Condition Page
* Refund Policy Page
