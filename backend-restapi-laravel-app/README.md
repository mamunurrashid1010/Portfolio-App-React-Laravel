# Portfolio App- Admin Panel
This is a admin panel of portfolio app and in this admin panel i developed using react-laravel-scaffolding.

### Module Overview <hr>
* Login
* Dashboard
* Services
    * Add New Service
    * All Service List 
    * Service Edit and Update
    * Delete Service
* Contact
    * All Contact List
    * Delete Contact List
* Sign Out

##### Rest Api Developed using Laravel.
